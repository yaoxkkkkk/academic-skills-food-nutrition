# Reporting Guidelines (EQUATOR + food/nutrition)

Match the study type to the right reporting checklist; require it in `food-paper`
Methods and check it in `food-review`. Systematic reviews use PRISMA; the
`systematic_reviewer` and `risk_of_bias` agents implement the detail.

## Core guidelines by study type
- **Randomized controlled trial** → **CONSORT** (+ trial registration, e.g. ISRCTN/ClinicalTrials.gov).
- **Observational (cohort/case–control/cross-sectional)** → **STROBE**.
- **Nutritional epidemiology** → **STROBE-nut** extension.
- **Systematic review / meta-analysis** → **PRISMA 2020** (+ PRISMA-ScR for scoping; PROSPERO registration).
- **Diagnostic/analytical accuracy** → **STARD**.
- **Animal studies** → **ARRIVE 2.0**.
- **In vitro** → clear method reporting; risk of bias via **OHAT** (default here).
- **Case reports** → **CARE**.
- **Qualitative (sensory/consumer)** → **COREQ / SRQR**.

## Food/nutrition specifics to report
- Food/matrix: source, cultivar/breed/batch, processing, storage.
- Composition: method (AOAC number), basis (wet/dry), g/100 g.
- Microbiology: medium, incubation, LOD, results as log CFU/g.
- Sensory: panel type, size, scale, screening/training, ethics/consent.
- Analytics: instrument conditions, LOD/LOQ, recovery, identification basis.
- Statistics: design, model, test, post-hoc, n (biological), effect size, threshold.
